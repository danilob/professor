Python 3 & Django 2.0

Resumo

O python tem se tornado uma linguagem de programação muito popular, em vários grupos de discussão discute-se sobre sua inserção no ensino de lógica de programação devido a facilidade em se programar nesta linguagem, sendo ela dinâmica, case sensitive, tipagem forte, conter muitas bibliotecas, documentação rica, entre outras qualidades como a obrigatoriedade da identação (código bem mais organizado). Para quem gosta de programar mas não perder tempo trabalhando com desenvolvimento de várias páginas web sem poder reutilizá-las o Django é uma ótima opção, um dos framewoks construídos para o python, que promove o desenvolvimento rápido e limpo para web. O estilo de arquitetura MVC (Model View Controller) vem a separar o programador/desenvolvedor e o designer promovendo uma velocidade no desenvolvimento de aplicações web, com suporte a banco de dados e inserção de elementos como estruturas condicionais no próprio template utilizando as tags do Django, entre outros recursos como a herança de templates. Para quem almeja programar para web mas não sabe por onde começar ou para quem quer conhecer outras formas de desenvolver para web: python e django é a mistura perfeita para se produzir muito com pouco código.
