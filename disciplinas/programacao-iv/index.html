Programação para Dispositivos Móveis

Informações gerais

Ementa: Estudo de uma linguagem de programação para desenvolvimento de aplicações para dispositivos móveis. Recursos, facilidades e aplicações que a linguagem escolhida oferece. Estudos de caso.

Objetivos: Entender as características da arquitetura, as ferramentas e as principais linguagens de desenvolvimento de aplicações para dispositivos móveis; e Obter embasamento teórico-prático das técnicas de desenvolvimento de sistemas para dispositivos móveis.

Instituição: Universidade Estadual do Piauí (UESPI).
Período: 2019.1.
Dia/horário: Qua e Qui - 10h20 às 12h e Sex - 7h às 8h40.
