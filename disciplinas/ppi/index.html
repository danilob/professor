Programação para Internet

Informações gerais

Ementa: Arquitetura da Internet e da World Wide Web (WWW). Modelo Requisição e Resposta. Linguagens de marcação e de estruturação de documentos de hipertexto. Geração de conteúdo dinâmico através de linguagens especializadas. Integração com Banco de Dados. Conceitos de servidor de aplicações e Web Services. Tecnologias para desenvolvimento de soluções Web 2.0.
Objetivos: Desenvolver sistemas web utilizando o paradigma orientado a objetos e baseados em componentes; Desenvolver aplicações web estruturado segundo o padrão MVC.
Instituição: Faculdade Lourenço Filho (FLF).
Período: 2018.2.
Dia/horário: Toda sexta-feira nos horários AB (18h30 - 20h20) e CD (20h25 - 22h15).
