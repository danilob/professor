Projeto de Extensão - Conhecendo o Latex

Informações gerais

Justificativa. Este projeto visa popularizar o uso do LaTeX para produção de textos acadêmicos científicos visando uma melhor qualidade e preparação para publicação em principais veículos de divulgação da ciência que utilizam esse sistema tipográfico em suas publicações, além de ser uma ferramenta para se adequar as regras exigidas pela ABNT. Atualmente poucos docentes e discentes de IES (Instituições de Ensino Superior) locais conhecem essa ferramenta que é conhecida nacional e internacionalmente.
Público-alvo. Docentes e discentes de IES da região de Parnaíba.
Localização. Campus Prof. Alexandre Alves de Oliveira (Parnaíba/PI)
Objetivos. Conhecer o sistema tipográfico LaTeX e utilizá-lo para construção de documentos eletrônicos acadêmicos científicos.
Metodologia. O principal método de ensino será concentrado em aulas expositivas, na troca de experiências e na aplicação imediata via computador, das técnicas lecionadas. Utilizaremos dois softwares para aplicação dos conhecimentos em LaTeX: o TeXStudio, software livre; e o Overleaf, editor de LaTeX online.
Avaliação. Os participantes do projeto serão avaliados de acordo com a participação ativa nas atividades dentro e fora do ambiente de ensino. Adotaremos também para fins de avaliar o projeto a aplicação de questionários.

Guia de Instalação do Latex e TexStudio

Instalação no Windows
  1. https://miktex.org/download → Selecione windows e baixe o arquivo executável para instalação.
  2. Após a instalação do miktex baixe o editor TexStudio. https://www.texstudio.org/ → clique no botão download now e instale-o em seu computador.
Instalação no Linux
  1. Abra o terminar (Crlt+T) e digite: sudo apt-get install texlive texlive-latex-extra texlive-lang-portuguese
  2. Baixe o editor TexStudio: https://www.texstudio.org/ → clique no botão download now e instale-o em seu computador.
Instalação no Mac
  1. Siga este guia para instalação dos pacotes.
  2. Baixe o editor TexStudio: https://www.texstudio.org/ → clique no botão download now e instale-o em seu computador.
