Projeto de Extensão - Conhecendo o Latex para TCC

Informações gerais

Justificativa. Este projeto visa popularizar o uso do LaTeX para produção de textos acadêmicos científicos visando uma melhor qualidade e preparação para publicação em principais veículos de divulgação da ciência que utilizam esse sistema tipográfico em suas publicações, além de ser uma ferramenta para se adequar as regras exigidas pela ABNT. Atualmente poucos docentes e discentes de IES (Instituições de Ensino Superior) locais conhecem essa ferramenta que é conhecida nacional e internacionalmente.
Público-alvo. Alunos do último bloco do curso de Computação - UESPI/Floriano.
Localização. Campus Dra. Josefina Demes (Floriano/PI)
Objetivos. Mostrar o LaTeX como ferramenta para edição do template de TCC.

Período: 2019.2.
Dia/horário 04/Novembro (à noite)

Slides e Materiais

Material Descrição Links
Slides Slides com informações gerais da estrutura que o LaTeX utiliza para criar excelentes documentos com qualidade tipográfica.
Quadro no Trello Neste material encontram-se de forma resumida e compacta boa parte do material necessário para se produzir os textos em LaTeX que serão utilizados no template de TCC.
Overleaf Editor de arquivos LaTeX online: ferramenta com acesso gratuito.
Template de TCC em LaTeX Template para TCC base com os requisitos determinados pelo NDE e Colegiado: Template baseado no template para artigos e dissertações da SBC com referências e citações baseado nas normas da ABNT.

Ferramentas Utéis

  • Editor online de equações LaTex
  • Gerador de Tabelas
  • Criação de referências em Bibtex a partir do ISBN
  • Google Scholar
  • Criação de referências em Bibtex a partir do DOI, PMCID, ou arXiv ID
  • Manutenção de Hiperlinks
